# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

- Nutmeg: hsl(14, 45%, 36%) color de los numeros
- Dark Raspberry: hsl(332, 51%, 32%) color del H3

### Neutral

- White: hsl(0, 0%, 100%) color del fondo de la tarjeta
- Rose White: hsl(330, 100%, 98%) color de fondo de la primera tabla
- Eggshell: hsl(30, 54%, 90%) fondo 
- Light Grey: hsl(30, 18%, 87%) color de las letras
- Wenge Brown: hsl(30, 10%, 34%) color del stronh
- Dark Charcoal: hsl(24, 5%, 18%)

## Typography

### Body Copy

- Font size (paragraph): 16px

### Fonts

- Family: [Young Serif](https://fonts.google.com/specimen/Young+Serif)
- Weights: 400

- Family: [Outfit](https://fonts.google.com/specimen/Outfit)
- Weights: 400, 600, 700
